Tests

During the development process, the modules will be required to pass more and more demanding tests. The tests will check whether the public APIs work as intended, but the developers are encouraged to write their own tests.

Tests will be split into development phases - V1, V2 and V3.

The tests code is meant to be LITERALLY PASTED INTO your testing code.

UavNav

V1
Take off, fly to a given point, land.
plane.takeoff(); time.sleep(30); assert plane.altitude > 10 plane.fly_to_point(point_lat, point_lon) plane.land() time.sleep(30) assert plane.altitude < 1
V2
Take off, fly to a given point, during the flight handle the hold() request (expect the plane to circle around the hold() zone in a distance lesser than 100m), wait for continue_flight() request, then land.
plane.takeoff(); time.sleep(30); assert plane.altitude > 10 plane.fly_to_point(point_lat, point_lon) time.sleep(100) plane.hold() plane_lat, plane_lon = plane.lat, plane.lon time.sleep(100) assert geog.distance((plane_lon, plane_lat), (plane.lon, plane.lat)) < 100 plane.continue_flight() time.sleep(10) plane.land() time.sleep(30) assert plane.altitude < 1
